Run the script as this:
./dom_parse_v2.py <input_file> 
eg.
./dom_parse_v2.py <output1a.txt>
Result
gid_pfam_def.txt file would be generated containing a TSV file of peptide_id, pFAM_id and Definitiion of PFAM_id.The output1a.txt is a file generated after running hmmscan with query FASTA file of sequences against Pfam-A dbase.
